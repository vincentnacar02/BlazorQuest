# BlazorQuest
Simple Text-based path following RPG 

### Use Chatgpt to generate quest story
provide below format in prompt

# JSON data quest format
```json
{
 "<path>": {
    "text": "<path text>",
    "options": [
      {
        "text": "<option text>",
        "goto": "<next path>"
      },
      {
        "text": "<option text>",
        "goto": "<next path>"
      }
    ]
  }
}
```
